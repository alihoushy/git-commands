آموزش دستورات گیت

دستور git init

git init

این دستور یک ریپازیتوری جدید گیت ایجاد میکنه، به این صورت که یه دایرکتوری جدید به نام git. ایجاد میشه که به عنوان ریپازیتوری گیت شناخته میشه و از این پس میتونید با گیت در این پروژه کار کنید.

git init DIRECTORY

این دستور مشابه دستور git init عمل میکنه با این تفاوت که ابتدا میاد و یه دایرکتوری به نام DIRECTORY میسازه و روند دستور git init رو در اون دایرکتوری انجام میده.

git init --initial-branch=BRANCH_NAME

این دستور مثل دستور git init عمل میکنه، اما با این تفاوت که یک برنچ اولیه به نام BRANCH_NAME نیز ایجاد میشه.

git init -b BRANCH_NAME

این دستور دقیقا مشابه دستور git init --initial-branch=BRANCH_NAME عمل میکنه اما با کدی کوتاه‌تر.

git init --bare

این دستور یه ریپازیتوری میسازه که نمیشه در اون فایلی رو ویرایش کرد یا تغییری رو کامیت کرد، تنها برای push و pull کردن کاربرد داره. به عنوان ریپازیتوری مرکزی (bare repository) نامگذاری میشه که به ریپازیتوری های non-bare متصل میشه.

دستور git config

git config user.name USERNAME

با این دستور، به گیت میگید که برای ریپازیتوری فعلیتون، یوزرنیم اکانت شما USERNAME هستش.

git config --global user.name USERNAME

با این دستور، به گیت میگید که برای تمام ریپازیتوری هایی که روی سیستم تون دارید، یوزرنیم اکانت شما USERNAME هستش.

git config user.email EMAIL

این دستور مشابه دستور git config user.name USERNAME هستش، با این تفاوت که ایمیل رو مشخص میکنه.

git config --global user.email EMAIL

این دستور مشابه دستور git config --global user.name USERNAME هستش، با این تفاوت که ایمیل رو مشخص میکنه.

git config user.password PASSWORD

این دستور مشابه دستور git config user.name USERNAME هستش، با این تفاوت که پسورد اکانت شما رو مشخص میکنه.

git config --global user.password PASSWORD

این دستور مشابه دستور git config --global user.name USERNAME هستش، با این تفاوت که پسورد اکانت شما رو مشخص میکنه.

git config user.signingkey SIGNING_KEY

با این دستور، به گیت اعلام میکنید که برای این ریپازیتوری، از کلید امضای GPG با مقدار SIGNING_KEY استفاده کن. قبل از استفاده از این دستور، باید کلید امضای GPG خودتونو بسازید.

git config --global user.signingkey SIGNING_KEY

با این دستور، به اعلام میکنید که برای تمام ریپازیتوری هایی که روی سیستم تون دارید، از کلید امضای GPG با مقدار SIGNING_KEY استفاده کن. قبل از استفاده از این دستور، باید کلید امضای GPG خودتونو بسازید.

git config init.defaultBranch BRANCH_NAME

با این دستور، برای گیت مشخص میکنید که برنچ پیش‌فرض ریپازیتوری فعلی تون که به تازگی اون رو init کردید، چی باشه.

git config --global init.defaultBranch BRANCH_NAME

با این دستور، برای گیت مشخص میکنید که برنچ پیش‌فرض برای ریپازیتوری های جدیدی که در آینده میسازید چی باشه.
